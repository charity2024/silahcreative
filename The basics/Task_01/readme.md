# Task 1: Semantic HTML – Cosmic Structure

**Roadmap Stage:** HTML (Semantic HTML, Accessibility, SEO basics)

## Project Goal
Build the clean HTML skeleton for a beautiful astronomy hero page. No styling yet — only structure.

## Design Direction
A full-screen hero page about exploring space through code.

## Requirements

### Must Have:
- Proper document structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Semantic HTML tags:
  - `<header>` for the top navigation
  - `<nav>` for menu
  - `<main>` for the main content
  - `<section>` for the hero area
  - `<h1>` and `<h2>` for headings
  - `<p>` for text
  - `<button>` for call-to-action
  - `<footer>` (simple)
- A container for stars (empty `<div class="stars"></div>`)
- Good accessibility:
  - `lang="en"` on html tag
  - Proper heading hierarchy
  - Alt text on any images (even if not added yet)
  - ARIA labels where necessary

### Content Ideas (Use these or make your own):
- Main Heading: "Exploring the Universe"
- Subheading: "One line of code at a time"
- Navigation links: Home, Discover, Gallery, About
- Button: "Begin Journey"

## Step-by-Step Instructions

1. Create a new folder: `cosmic-basics`
2. Inside it create `index.html`
3. Write the full HTML structure
4. Make the page have a black background using inline style for now (just so you can see it)
5. Open the file in your browser and check how it looks

## Success Criteria
- The HTML is clean and well-structured
- All content is visible
- You used semantic tags instead of only `<div>` everywhere
- Page has good hierarchy (H1 > H2 > paragraphs)
