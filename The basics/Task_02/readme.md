# Task 2: CSS Layouts & Animations – Cosmic Styling

**Roadmap Stage:** CSS + CSS Animations + Making Layouts (Responsive, Pixel Perfect, Animated)

## Project Goal
Take the HTML structure you built in Task 1 and transform it into a beautiful, cinematic cosmic hero page using only CSS.

## What You Will Practice
- CSS selectors and specificity
- Box model, positioning, flexbox
- Responsive design (mobile + desktop)
- Advanced CSS: gradients, shadows, glows
- CSS Animations with @keyframes
- Pixel-perfect attention to detail

## Requirements

### 1. Layout & Styling
- Make the hero section full screen (`100vh`)
- Perfectly center the text both horizontally and vertically
- Style the navigation bar (top or bottom)
- Beautiful cosmic color scheme:
  - Background: Deep black / dark navy gradient
  - Text: White with soft glow
  - Accents: Soft purple, cyan, or gold

### 2. Visual Enhancements
- Add multiple layers of stars using CSS only
- Create a subtle nebula/gradient background
- Add glow effects to text and button
- Style the button to look cosmic (hover effect required)

### 3. CSS Animations (Key Part)
- **Twinkling Stars** — create at least 50–100 small stars that twinkle at different speeds and delays using `@keyframes`
- **Text Animation** — fade in + slight upward movement on page load
- **Button Hover** — scale + glow + color change
- **Floating Elements** — make one or two planets or small elements float slowly

### 4. Responsive Design
- Text and spacing must look good on mobile phones
- Navigation should adapt (stack or hide on small screens)

## Step-by-Step Instructions

1. Use the same `index.html` from Task 1
2. Create (or update) `styles.css` and link it in your HTML
3. Style the page section by section:
   - First: Overall page + background
   - Second: Navigation
   - Third: Hero text and button
   - Fourth: Stars container
4. Add CSS animations
5. Make it responsive using media queries

## Success Criteria
- The page feels cinematic and space-like
- Stars are twinkling smoothly
- Everything is centered and balanced
- Looks good on both desktop and mobile
- No layout breaking

## Tips for Beginners
- Use Chrome DevTools (Right click → Inspect) a lot
- Google things like:
  - “CSS twinkling stars”
  - “CSS full screen hero”
  - “CSS glow text”
  - “CSS floating animation”
- Start with layout first, then add animations
- Keep your CSS organized with comments

## Bonus (Do if you feel confident)
- Add a very slow moving background gradient (subtle nebula effect)
- Make stars different sizes and colors (white, light blue, yellow)
